# go-http-server

## ディレクトリ

- ./go

## 準備

- ./go/env/.envファイルを作成（./env/.env.sampleをベースに）

## 実行

```shell
# ローカル開発などコード改変を実施する場合
make run

# バイナリをビルドして挙動確認（環境変数は反映されない）
make build

# goテスト実行
make test
```
